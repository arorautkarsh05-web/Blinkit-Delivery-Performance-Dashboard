# Blinkit Delivery Performance Dashboard

## 📌 Project Overview
This project presents an interactive Power BI dashboard for monitoring Blinkit's delivery performance. The dashboard helps analyze delivery efficiency, identify delay reasons, and monitor delivery trends.

## 🎯 Objectives
- Track total orders
- Monitor average delivery time
- Analyze average delivery distance
- Measure on-time deliveries
- Identify major delay reasons
- Analyze monthly delivery trends

## 📊 KPIs
- Total Orders
- Average Delivery Time
- Average Distance
- On-Time Orders

## 📈 Dashboard Visualizations
- Delivery Status Distribution (Donut Chart)
- Delay Reasons (Bar Chart)
- Monthly Deliveries Trend (Line Chart)
- Average Distance by Delivery Status (Column Chart)

## 🎛 Interactive Features
- Delivery Status Filter
- Delay Reason Filter
- Date Filter

## 🛠 Tools Used
- Microsoft Power BI
- Microsoft Excel

## 📂 Dataset
Blinkit Delivery Performance Dataset
